生涯規劃

學習計畫與營期/營後目標。

營期間學習計畫

以穩定的節奏推進進度：白天專注學習與實作，晚上以紀錄與反思收尾； 透過公開紀錄建立紀律，也讓自己持續得到回饋並調整方向。

  • 一～五：每日 8–10 小時（上課、實作、作業/檔案作業、記錄與回顧）
  • 六：複盤 & 周回顧，將整周的學習重點發佈到 Threads
  • 日：休息日/補足前幾天落下的進度

營期間學習目標

以後端為核心，從資料庫與 API 開發為起始點，逐步建立容器化與雲端部署的基本功； 同時在專題中培養協作能力與發 PR 的好習慣。

營期後學習目標

營期結束後，將專注在系統化整理與證照認證，補足型別與雲端技能， 以便在求職階段展現可維護、可上線的作品與穩定的工程素養。
